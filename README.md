# Tuberculosis Detection from X-rays (MLOps Project)

## Overview
This project detects TB from chest X-ray images using deep learning.

## Problem Type
Binary Classification:
- TB Positive
- TB Negative

## Features
- Deep learning-based image classification
- REST API using FastAPI
- Modular code structure
- Ready for Docker deployment

## Tech Stack
- Python
- TensorFlow / PyTorch
- FastAPI
- Docker (planned)

Datasets are not included due to size and privacy.
